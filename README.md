## Team Service


```http
Name Methode
```

| Key      | Description                |
| :-------- | :------------------------- |
| **'Set'**          | Assigns or updates a value in a variable or object|
|**'Get'**| Retrieves the current value from a variable or object|
|**'Remove'**|Deletes an element or property from a collection or object|
